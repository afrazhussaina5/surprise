# Hospital Management System

## Project Description
The Hospital Management System is a web-based front-end application designed to manage basic hospital operations such as patient registration, appointment booking, medicine inventory, and patient tracking. This project focuses on user interface design and navigation using modern web technologies.

---

## Features
- Login system (UI-based)
- Dashboard with hospital statistics
- Patient registration form
- Appointment booking module
- Medicine inventory management
- Patient detail tracking
- Responsive and user-friendly design

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
